Speciality IO

The FPGA Addon supports common speciality functions. It contains the IP cores that can be added to you FPGA application in:

\{Public Documents}\National Instruments\NI VeriStand {Year}\Custom Devices\FPGA Addon\FPGA IP.

The supported functions are:

  • PWM Generation (PWM.Generate.Digital.vi)
  • Sinewave Generation (Sine.Generate.vi)
  • Analog PWM Measurement (PWM.Measure.Analog.vi)
  • Digital PWM Measurement (PWM.Measure.Digital.vi)

PWM Generation

Allows user to output a digital PWM signal. Parameters are:
  • Duty Cycle: Enter from 0 to 100%
  • Frequency: Enter Hz
  • Active Edge: 0 = Low, 1 = High

Sinewave Generation

Allows user to output an analog sinewave signal. Parameters are:
  • Frequency: Enter Hz
  • Sinewave Offset: Enter from +/- 128
  • Amplitude: Peak to Peak from +/- 128

Analog PWM Measurement

Allows user to measure an analog PWM signal. Parameters are:
  • Threshold Level: This value is compared against the analog PWM signal to determine if the signal is high or low. Enter from +/- 128.
  • Timeout: The number of ticks of the owning FPGA loop to acquire without seeing a level transition before reseting the outputs.
  • Active Edge: 0 = Low, 1 = High

Results from this core are:

  • Frequency: In Hz
  • Period: In Seconds
  • Duty Cycle: In Percent (%)
  • Active Time: In Seconds
  • Active Voltage: Maximum voltage level during the active edge of the measurement. NOTE: If the active edge is low (0), then the minimum voltage level is used.
  • Inactive Voltage: Minimum voltage level during the inactive edge of the measurement. NOTE: If the active edge is low (0), then the maximum voltage level is used.

Digital PWM Measurement

Allows user to measure a digital PWM signal. Parameters are:
  • Timeout: The number of ticks of the owning FPGA loop to acquire without seeing a level transition before reseting the outputs.
  • Active Edge: 0 = Low, 1 = High

Results from this core are:

  • Frequency: In Hz
  • Period: In Seconds
  • Duty Cycle: In Percent (%)
  • Active Time: In Seconds
