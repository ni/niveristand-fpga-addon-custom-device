Scalar Outputs

This section contains all the available controls of the supported data types in the FPGA bitfile.

What do you want to do?

Add an Output

Select the channel or channels you would like to add from the Available Registers list. Then select Add Selected.

Remove an Output

Select the channel or channels you would like to remove from the Available Registers list. Then select Remove Selected.

Enable Channel Bitpacking

Any integer datatype channel (ie U8, I32, etc) can be converted to a bitpacked channel. This bitpacked channel allows the user to convert an array of Booleans to an integer. To convert the channel to bitpacked, right click on the channel and select Enable Bitpacked. Note: if the right click option is not available, it means the channel is a Boolean or SGL and does not support bitpacking.

Remove Channel Bitpacking

Right click on the channel and select Disable Bitpacked.
